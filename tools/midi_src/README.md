# MIDI sources (owner-supplied, 2026-06-10)

Anonymous/unlabeled-source MIDI encodings of two public-domain compositions
(Debussy "Golliwogg's Cake Walk" 1908; Fučík "Entry of the Gladiators" 1897),
supplied by the owner with the licensing call accepted (PD compositions;
note-data provenance unlabeled). Rendered to the game tracks via fluidsynth +
FluidR3_GM (MIT): golliwog -> Calliope 82 / Reed Organ 20 split at MIDI 58;
gladiators -> Honky-tonk 3 (tack piano). The earlier clean-room transcription
generators (gen_golliwog_calliope.py / gen_gladiators_tackpiano.py) are kept
as the zero-risk fallback.
